## Clonar el Repositorio

Para obtener una copia local del proyecto, clona el repositorio:

```bash
git clone https://github.com/usuario/proyecto.git
Crear una Rama Nueva
Crea una nueva rama para trabajar en tus modificaciones:

bash
Copiar código
git checkout -b nombredelarama
Ir a la Rama
Verifica que estás en la rama correcta:

bash
Copiar código
git branch nombredelarama
Añadir las Modificaciones a la Rama
Añade los archivos modificados al área de preparación:

bash
Copiar código
git add .
Commit con Descripción de lo Modificado
Haz un commit con una descripción clara de los cambios realizados:

bash
Copiar código
git commit -m "Descripción del cambio realizado"
Subir los Cambios a la Rama
Sube tus cambios al repositorio remoto en la rama correspondiente:

bash
Copiar código
git push origin nombredelarama
Crear un Pull Request
Desde GitHub, abre un Pull Request para fusionar tu rama con main:

main <- nombredelarama
Actualizar tu Rama con la Rama Principal
Antes de fusionar los cambios a main, asegúrate de que tu rama esté actualizada:

bash
Copiar código
git checkout main
git pull origin main
git checkout nombredelarama
git merge main
Fusionar la Rama
Una vez aprobado el Pull Request, fusiona tu rama con la rama principal:

bash
Copiar código
git checkout main
git merge nombredelarama
Copiar código

Este bloque incluye todos los pasos en un formato limpio y fácil de seguir.
